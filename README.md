🍔 Sistema de Gestión Empresarial - McDonald's

Sistema de escritorio desarrollado en Java utilizando Apache NetBeans y
Oracle Database,orientado a la gestión empresarial de un restaurante McDonald's.

---

🚀 Características principales

- 📦 Gestión de productos
- 👥 Gestión de clientes
- 🧾 Facturación
- 💳 Métodos de pago
- 🏢 Gestión de sucursales
- 📊 Reportes del sistema
- 👨‍💼 Gestión de empleados
- 🛒 Control de pedidos
- 📁 Panel administrativo moderno


🛠️ Tecnologías utilizadas

- Java
- Apache NetBeans
- Oracle Database
- Swing GUI
- JDBC
- Git & GitHub
